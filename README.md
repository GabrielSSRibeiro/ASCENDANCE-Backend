## Technologies
This project was developed with the following:
- Node.js
- React JS
- MongoDB
- REST API
- WebSocket protocol
- Responsive UI
- MVC architecture 
- Atomic Design pattern

<br>

## Project
Web application to create games and characters in an original RPG system.

<br>

## Getting Started
- Install node.js
- Run npm install
- To run frontend: yarn Start
- To run backend: yarn dev
